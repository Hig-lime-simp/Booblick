## Описание проекта

«Бубликометр» — это шуточное приложение, которое определяет, насколько вы «бублик» в процентах.  
Процент рассчитывается на основе трёх параметров:

- Имя
- Количество часов сна
- Количество часов в видеоиграх

Если пользователь вводит **Имя = Гриша**, **Сон = 2**, **Игры = 22** — он получает 100% бубличности.  
Совпадение одного из условий даёт 33%, двух — 66%, трёх — 100%.

## Запуск проекта

### Требования
- Flutter SDK (≥3.0)
- Dart SDK (≥3.0)

### Инструкция

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Hig-lime-simp/bublik_meter.git
   ```
2. Перейти в папку проекта:
   ```bash
   cd booblick_project
   ```
3. Установить зависимости:
   ```bash
   flutter pub get
   ```
4. Запустить приложение:
   ```bash
   flutter run
   ```
